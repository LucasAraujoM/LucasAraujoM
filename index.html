Soy desarrollador backend especializado en PHP y Laravel, con experiencia construyendo APIs y aplicaciones web escalables orientadas a resolver problemas reales de negocio. Actualmente trabajo como Full Stack Developer de forma freelance, donde diseño e implemento soluciones completas: desde la arquitectura backend y modelado de datos hasta la integración de frontend y despliegue. Proyectos

EmailChecker

Herramienta diseñada para validar la salud y entrega potencial de direcciones de correo electrónico. Ofrece análisis de formato, verificación de dominios, verificación de servidor de buzón (MX) y conexión SMTP para confirmar que el buzón existe realmente, reduciendo rebotes y mejorando la calidad de las campañas de email.

PHP, Laravel, API REST

Barber CRM (Desktop/Web)

Sistema de gestión de barbería diseñado para optimizar la experiencia tanto del cliente como del profesional. Permite agendar citas de manera eficiente, gestionar el flujo de trabajo y mantener un registro detallado de clientes y servicios. Facilita la administración diaria de la barbería, mejora la organización de la agenda y ayuda a fidelizar clientes mediante un servicio más profesional y personalizado.

PHP, Laravel, MySQL, Livewire, NativePHP

Monitor de Gastos (Mobile)

Herramienta diseñada para el seguimiento y análisis de gastos personales. Permite gestionar ingresos y egresos, categorizar movimientos, establecer presupuestos y visualizar patrones de gasto a través de gráficos intuitivos. Su objetivo es proporcionar una visión clara de la salud financiera personal, ayudando a tomar decisiones informadas para un mejor control del dinero.

PHP, Laravel, MySQL, Filament, NativePHP

My Uptime

Plataforma diseñada para monitorear el estado de sitios web, APIs, servidores y aplicaciones, enviando alertas inmediatas cuando detecta caídas o problemas de rendimiento. Permite configurar intervalos de chequeo, notificaciones personalizadas y consultar historial de disponibilidad. Su objetivo es ayudar a mantener la continuidad del servicio, asegurando que los servicios estén siempre operativos y detectando incidentes rápidamente.

PHP, Laravel, API REST, MySQL, Filament
